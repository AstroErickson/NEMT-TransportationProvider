## Summary of Project 

React PDF Form Generator

This project demonstrates a React web application that collects form data and generates a PDF document, including uploaded files.

The application allows users to complete a form and upload supporting documents in TXT or PDF format. After submission, the application dynamically generates a PDF document containing the form data and concatenates the uploaded files into the final PDF output.

This project demonstrates how to:

Build a dynamic React form

Handle file uploads (TXT and PDF)

Process uploaded files in the browser

Generate PDFs programmatically

Merge multiple PDFs and text files into a single document

The final generated document contains:

Structured form data

Uploaded text files converted to PDF pages

Uploaded PDF files merged into the final document

This project serves as an example of how client-side React applications can process uploaded files and produce combined documents without requiring a backend service.
